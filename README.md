# 🚀 Personal Portfolio — Web Developer

A modern, animated, and performance-focused personal portfolio built to showcase frontend skills, UI craftsmanship, and motion design thinking.

This portfolio is designed to be **recruiter-friendly**, **interview-ready**, and visually engaging without being distracting.

## ✨ Highlights

- 🎨 Clean, modern UI with light & dark mode
- 🌌 Subtle animated background for visual depth
- ⚡ Smooth animations using **Framer Motion** and **GSAP**
- 🧠 Thoughtful frontend architecture (data-driven sections)
- 📄 Downloadable CV / Resume
- 📊 Contact form integrated with Google Sheets
- 📱 Fully responsive across devices

---

## 🛠 Tech Stack

**Frontend**
- React
- JavaScript (ES6+)
- Tailwind CSS

**Animations**
- Framer Motion (UI & component animations)
- GSAP (advanced & continuous animations)

**Data & Integrations**
- Google Sheets (contact form data storage)

**Tooling**
- Vite
- Git & GitHub
- Lucide Icons

---

## 🧩 Architecture Overview

```txt
src/
├─ components/        # Reusable UI components
├─ pages/             # Page-level components
├─ data/              # Centralized data (skills, projects, nav)
├─ animations/        # Reusable motion variants
├─ layouts/           # Layout wrappers
├─ styles/            # Global styles
└─ utils/             # Helper utilities
```
